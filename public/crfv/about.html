User Guide

1 Getting Started

To begin using the CRFV Event System, follow these initial setup steps:

  • Log in using your administrator credentials
  • Configure your organization settings under Profile > Settings
  • Add team members and assign roles
  • Set up your default event template

2 Creating an Event

Create a new event in just a few steps:

  • Navigate to the "Create Event" page
  • Enter event details: name, date, location, description
  • Add sessions, speakers, and sponsors
  • Configure registration settings and ticket types
  • Publish your event to make it visible to participants

3 Managing Registrations

Efficiently handle participant registrations:

  • Use the "Register" page to add participants manually
  • Import participants from CSV/Excel files
  • Set up automated confirmation emails
  • Track registration status and send reminders
  • Manage waitlists for popular events

4 Tracking Attendance

Streamline your check-in process:

  • Use the "Attendance" page for real-time check-in
  • Scan QR codes from registration emails
  • Search participants by name for manual check-in
  • View attendance status for each session
  • Generate attendance certificates

5 Generating Reports

Access valuable insights from your event data:

  • Navigate to the "Reports" section
  • Select from pre-built report templates
  • Customize date ranges and filters
  • Export reports to Excel, PDF, or CSV
  • Schedule automatic report delivery

Frequently Asked Questions

How do I reset my password? expand_more

If you've forgotten your password, click the "Forgot Password" link on the login page. You'll receive an email with instructions to reset your password. If you don't receive the email within 5 minutes, check your spam folder or contact support.

Can I customize the registration form? expand_more

Yes! In the event creation settings, navigate to the "Registration" tab where you can add custom fields, change field order, make fields required/optional, and even create conditional fields that appear based on previous answers.

How many participants can I register for an event? expand_more

There is no set limit to the number of participants you can register. For events with more than 5,000 attendees, we recommend contacting our support team for performance optimization tips.

Can I integrate with other calendar systems? expand_more

Yes, CRFV supports integration with Google Calendar, Outlook Calendar, and Apple Calendar. You can enable calendar integration in your account settings under "Integrations".

How secure is my event data? expand_more

Security is our top priority. We use bank-level 256-bit encryption for all data transmission and storage. Regular security audits, two-factor authentication, and role-based access controls ensure your data remains protected.
