Tailwind CSS Proficiency Quiz

Multiple Choice Questions (20 items)

1

What is the primary philosophy behind Tailwind CSS?

Correct Answer: B. Utility-first approach

Tailwind is a utility-first CSS framework, meaning it provides low-level utility classes to build designs directly in your markup.

2

Which of these classes would add 1rem (16px) of padding to all sides of an element?

Correct Answer: B. p-4

In Tailwind, p-4 adds 1rem (16px) padding to all sides. The number represents multiples of 0.25rem (4 = 1rem).

3

How would you make a flex container that aligns items vertically in the center?

Correct Answer: A. flex items-center

items-center aligns flex items along the cross axis (vertically for row direction). justify-center would align along the main axis (horizontally for row direction).

4

Which Tailwind class would you use to make text bold?

Correct Answer: C. font-bold

Tailwind uses font-bold for bold text (font-weight: 700). Other options include font-normal, font-semibold, and font-extrabold.

5

What does the "md:" prefix indicate in Tailwind classes?

Correct Answer: B. Medium screen size breakpoint

The md: prefix applies styles at the medium breakpoint (768px by default). Tailwind uses mobile-first breakpoints: sm:, md:, lg:, xl:, 2xl:.

6

Which Tailwind class would you use to create a subtle shadow effect?

Correct Answer: B. shadow

Tailwind provides multiple shadow sizes: shadow-sm (smallest), shadow (default), shadow-md, shadow-lg, shadow-xl, and shadow-2xl (largest).

7

How would you set a dark blue background color using Tailwind's default palette?

Correct Answer: A. bg-blue-800

Tailwind's color system uses numerical suffixes from 50-900, where higher numbers are darker. bg-blue-800 is a dark blue in the default palette.

8

Which class would add 2rem (32px) of margin to the top of an element?

Correct Answer: A. mt-8

Tailwind's spacing scale uses multiples of 0.25rem (4px by default). mt-8 adds margin-top of 2rem (8 Ã— 0.25rem). m-8 would add margin to all sides.

9

What does the "dark:" prefix do in Tailwind?

Correct Answer: B. Applies styles when dark mode is active

The dark: variant applies styles when dark mode is enabled in the operating system or when manually toggled. Example: dark:bg-gray-800.

10

Which utility would you use to make an element take up the full width of its container?

Correct Answer: C. w-full

w-full sets width to 100% of the parent container. w-screen would make it 100vw (viewport width), which might cause horizontal scrolling.

11

How would you create a grid with three equal-width columns?

Correct Answer: A. grid grid-cols-3

The grid class establishes a grid container, and grid-cols-3 creates three equal-width columns. Tailwind supports up to 12 columns by default.

12

Which class would center text horizontally?

Correct Answer: A. text-center

text-center centers text horizontally. justify-center is for flex/grid alignment, and content-center aligns grid content.

13

What does the "lg:" breakpoint represent by default?

Correct Answer: C. 1024px

Tailwind's default breakpoints are: sm: 640px, md: 768px, lg: 1024px, xl: 1280px, 2xl: 1536px.

14

How would you add rounded corners to an element?

Correct Answer: A. rounded

rounded gives a border-radius of 0.25rem (4px). You can use sizes like rounded-md (0.375rem), rounded-lg (0.5rem), or specific corners like rounded-t-lg.

15

Which class would make an element have absolute positioning?

Correct Answer: B. absolute

Tailwind uses direct CSS property names for positioning: absolute, relative, fixed, sticky. You'll typically combine with top-0, right-0, etc.

16

How would you create a hover effect that changes text color to red?

Correct Answer: A. hover:text-red-500

State variants like hover:, focus:, and active: can be prefixed to many utility classes. The syntax is always variant:utility-value.

17

Which class would set the font size to 1.25rem (20px)?

Correct Answer: C. text-lg

Tailwind's default font sizes: text-sm (0.875rem), text-base (1rem), text-lg (1.25rem), text-xl (1.5rem), up to text-9xl.

18

How would you create a container with a max-width of 1280px?

Correct Answer: A. container

The container class centers your content with max-width at each breakpoint (1280px at xl). For fixed widths, use max-w-7xl (80rem) or similar.

19

Which class would add a 1px solid gray border?

Correct Answer: A. border border-gray-300

Border styling requires two classes: border for width/style and border-{color} for color. You can adjust width with border-2, border-4, etc.

20

How would you make an element invisible but still take up space?

Correct Answer: B. invisible

invisible sets visibility: hidden (takes space), while hidden sets display: none (removes from flow). opacity-0 makes transparent but remains interactive.

True or False Questions (10 items)

21

Tailwind CSS uses a mobile-first approach for responsive design.

Correct Answer: True

Tailwind follows a mobile-first breakpoint system where unprefixed utilities take effect on all screen sizes, while prefixed utilities (like md:) only take effect at that breakpoint and above.

22

The hover: variant can only be used with color-related classes.

Correct Answer: False

The hover: variant can be used with many utility classes, not just colors. For example: hover:bg-blue-500, hover:shadow-lg, hover:scale-110, etc.

23

Tailwind automatically removes all unused CSS in production builds.

Correct Answer: True

Tailwind uses PurgeCSS (now part of PostCSS) to analyze your HTML/JS and remove any unused utility classes in production, resulting in very small CSS files.

24

You can customize Tailwind's default configuration by editing the tailwind.config.js file.

Correct Answer: True

The tailwind.config.js file allows extensive customization of colors, spacing, fonts, breakpoints, and more. You can extend or override the default theme.

25

Tailwind includes pre-built components like navbars and cards out of the box.

Correct Answer: False

Tailwind provides utility classes to build components, not pre-made components. However, official UI libraries like Tailwind UI offer pre-built components.

26

The `focus:` variant is used for styling elements when they have keyboard focus.

Correct Answer: True

The `focus:` variant applies styles when an element receives focus, crucial for accessibility. Example: `focus:ring-2 focus:ring-blue-500`.

27

Tailwind's spacing scale is based on multiples of 0.25rem (4px by default).

Correct Answer: True

Tailwind uses a spacing scale where 1 unit = 0.25rem (4px with default settings). So `p-4` = 1rem (16px), `m-2` = 0.5rem (8px), etc.

28

You need to use a build tool like webpack to use Tailwind CSS.

Correct Answer: False

While build tools are recommended for production, you can use Tailwind via CDN for basic projects. However, you lose customization and purge capabilities.

29

The `active:` variant styles elements when they are being clicked.

Correct Answer: True

The `active:` variant applies styles while an element is being activated (mouse down). Useful for button press effects. Example: `active:scale-95`.

30

Tailwind was created by the same team that created Bootstrap.

Correct Answer: False

Tailwind was created by Adam Wathan, Jonathan Reinink, David Hemphill, and Steve Schoger. Bootstrap was created by Twitter developers Mark Otto and Jacob Thornton.
